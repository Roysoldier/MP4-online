Retrouvez ici un ensemble d'éléments vous permettant d'utiliser le robot Lego Mindstorms EV3.

Voici une liste des outils à votre disposition :

  • NOGOPROG est un outil qui vous permet de créer votre propre programme pour votre robot
  • Les programmes tout prêt vous permettent de lancer le robot sans vous soucier de faire votre propre programme
  • L'aide vous explique comment procéder pour importer votre programme ou un que vous télécharger sur notre site, dans le robot Lego Mindstorms EV3.


Application Windows :

Nous vous proposons également une application Windows développée par Hugo qui permet de consulter le site internet via la connexion invité !

Vous pouvez la télécharger ici ou voir des photos du code et de la conception ici.


A propos de ce projet :

Ce projet à été conçu par Noan, Anaïs et Hugo dans le cadre du cours de NSI de terminale.

Pour ce dernier projet de l'année, nous avons décidé de nous donner au maximum pour vous offrir une expérience de qualité : Programmes tous prêts, conception de votre propre programme, fonctionnalités avancées et ajout de boutons personnalisés sur le robot, sans oublier une aide pour vous permettre de tester le robot sans difficulté. Nous avons aussi pensé à vous laisser la possibilité de créer un compte sur notre site afin que vous puissiez enregistrer les programmes que vous pourriez concevoir.

Après un bon paquet d'heures de travail, nous sommes fiers de vous présenter ce site avec toutes ces fonctionnalités.

Si vous rencontrez des bugs ou avez des questions, n'hésitez pas à nous contactez via ce formulaire.

L'équipe NOGO vous souhaite une bonne journée.
