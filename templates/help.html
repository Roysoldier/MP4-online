Page d'aide à l'installation d'un programme sur l'EV3 via VS Code

      1-   Lancer un programme prédéfini :

Si vous souhaitez lancer un programme prédéfini, nous vous invitons à les retrouver dans le robot via le menu "file browser".

Si malheureusement, le programme que vous souhaitez n'est pas dans le robot, nous vous invitons à le télécharger au format ".zip" dans la page des programmes sur notre site internet, puis de suivre les instructions de ce document en important le dossier que vous aurez téléchargé au préalable.


      2-   Lancer un programme que vous avez conçu :

Pour lancer un programme que vous avez conçu, accéder tout d'abord à l'éditeur NOGOPROG puis, cliquez sur le bouton enregistrer.

Ensuite, télécharger cette archive, ele contient les fichiers minimums pour que le robot soit fonctionnel. Extrayez les fichiers de l'archive et ajoutez y votre programme.

Enfin, ouvrez Visual Studio Code et téléversez le programme comme dans l'option 1.


      3-   Bugs, problèmes et solutions :


Problème Solution
Le programme ne se lannce pas ; il s'arrête avant d'avoir démarré Vérifiez la connexion des fils et les débrancher et les rebrancher au besoin. Nous avons remarqué qu'il y avait des faux-contacts. Vous pouvez aussi voir les erreurs via VS Code.
Impossible de connecter le robot via VS CodeVérifiez que les câbles sont correctement branchés et vérifier que VS Code et l'EV3 peuvent communiquer à travers le pare-feu
